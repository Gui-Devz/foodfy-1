# :rocket: Features

### Main features

* About page
* Recipes page
* Recipe description page
  * Show/hide sections

### Plus features
* Fully **responsive**
* **Dark mode** theme

# :runner: Getting Started

### **1. Clone repository**

```git clone git@github.com:rpradosilva/foodfy.git```

### **2. Install dependencies**

```npm install```

### **3. Run project**

```npm start```
